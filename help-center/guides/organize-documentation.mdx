## Overview

Set up your project docs for easy navigation.

<Steps>
  <Step title="Create key sections">
    Make pages for main areas: Overview, Setup, Usage, FAQ.
  </Step>
  <Step title="Add pages">
    Click "Add Page," enter a title, and add your content. Use markdown or the editor.
  </Step>
  <Step title="Arrange order">
    Drag and drop pages in the sidebar to set the best order for users.
  </Step>
  <Step title="Share access">
    Use the "Invite" button on your dashboard to add team members.
  </Step>
</Steps>
