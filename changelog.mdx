## Upgrade Instructions

Follow these steps to upgrade Саша Фелюгин to the latest version. Always back up your data first.

<Steps>
  <Step title="Backup Your Data" icon="database">

  Export your projects via the dashboard at `https://dashboard.example.com/export`.

  </Step>
  <Step title="Update Package" icon="package">

  <CodeGroup tabs="npm,yarn">
  ````bash
  npm install sasha-felyugin@latest
  ````
  ````bash
  yarn add sasha-felyugin@latest
  ````
  </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="settings">

  If upgrading from v1.0.0, run the migration script:

  ````bash
  npx sasha-felyugin migrate
  ````

  </Step>
  <Step title="Restart Service" icon="refresh-cw">

  Restart your application server and verify at `https://dashboard.example.com`.

  </Step>
</Steps>

<Callout kind="alert" title="Breaking Changes">
  v1.2.0 introduces config schema updates. Review the [migration guide](#) for details on `{oldConfig}` to `{newConfig}` changes.
</Callout>
