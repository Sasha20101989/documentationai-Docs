## Prerequisites

<Callout kind="info" title="Before You Begin">
You need a modern web browser like Chrome, Firefox, or Safari, and a valid email address. No software installation required.
</Callout>

## Create Your Account

Sign up quickly to access your dashboard.

<Steps>
  <Step title="Visit the Signup Page" icon="external-link">
    Open your browser and go to `https://sashafelyugin.com/signup`.
  </Step>

  <Step title="Enter Your Details" icon="user">
    Provide your email, create a strong password, and complete the CAPTCHA if prompted.
  </Step>

  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification link from Саша Фелюгин and click it to activate your account.
  </Step>
</Steps>

## Navigate the Dashboard

After login, you land on the main dashboard at `https://app.sashafelyugin.com/dashboard`. Use the sidebar to access spaces, templates, and settings. The top navigation shows your recent activity and search bar.

<Callout kind="tip">
Pin frequently used spaces to the dashboard for quick access.
</Callout>

## Set Up Your First Doc Space

Create a dedicated space for your project docs.

<Steps>
  <Step title="Create New Space" icon="plus">
    Click **New Space** in the dashboard and name it, e.g., `My Project Docs`.
  </Step>

  <Step title="Add Pages" icon="file-text">
    Inside the space, click **New Page** to start with a blank page or select a template.
  </Step>

  <Step title="Organize Structure" icon="layout">
    Drag and drop pages in the sidebar to build your hierarchy.
  </Step>
</Steps>

## Basic Editing and Publishing

Edit content using familiar tools and publish instantly.

<Tabs>
  <Tab title="Markdown Editor" icon="code">
    Use Markdown for precise control.

````markdown
# Welcome to My Docs

## Features

- Fast editing
- Real-time previews

> This is a blockquote.
````

    Preview updates live as you type.
  </Tab>

  <Tab title="Visual Editor" icon="edit-3">
    Drag-and-drop blocks for non-coders.

    Add headings, lists, images, and code blocks via the toolbar. Toggle to Markdown view anytime.
  </Tab>
</Tabs>

Click **Publish** to make changes live. Share via public URL or invite collaborators.

## Import Existing Content

Bring in docs from other platforms seamlessly.

<CodeGroup tabs="cURL,JavaScript">
````bash
curl -X POST https://api.sashafelyugin.com/v1/import \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "source": "github",
    "repo": "user/repo",
    "branch": "main"
  }'
````

````javascript
const response = await fetch('https://api.sashafelyugin.com/v1/import', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    source: 'notion',
    workspace: 'your-workspace-id'
  })
});
````
</CodeGroup>

<Expandable title="Advanced Import Options" default-open="false">
Use the dashboard importer for GitHub, Notion, or Google Docs. Map pages automatically during import.
</Expandable>
